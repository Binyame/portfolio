What is McCan - Creative Personal Portfolio Template

McCan is a multipurpose personal, portfolio and resume template. It’s creative, minimal and clean design. You can use it for, corporate and portfolio, freelancers, business, photo studio, portfolio theme and creative agency etc. This is a fully customizable template. You can edit each and every part of this template according to your needs.
